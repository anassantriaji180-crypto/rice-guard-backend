3. REFRESH & NEW SCAN SYSTEM
================================================================================

Two-Button Control System:

🔄 REFRESH ANALYSIS
  • Re-analyzes the current image
  • Maintains captured photo
  • Updates predictions
  • Use case: Verify results, try different models

📸 NEW SCAN
  • Clears all results and predictions
  • Resets UI to initial state
  • Allows fresh image upload or camera session
  • Use case: Quick restart without page reload

Functions:
  • resetScanner()    - Clear results only
  • startNewScan()    - Full reset
  • stopCamera()      - Close camera stream

9. CAMERA WORKFLOW
================================================================================

Step 1: Initialize
  User taps "📷 Start Camera" button
  ↓
Step 2: Request Permission
  Browser shows camera permission dialog
  ↓
Step 3: Stream
  Video feed displays with LIVE indicator
  ↓
Step 4: Capture
  User clicks "📸 Capture" button
  Canvas captures current frame
  ↓
Step 5: Classify
  Image sends to Flask backend
  YOLOv8 model processes
  ↓
Step 6: Display Results
  Disease classification displayed
  Refresh controls appear
  ↓
Step 7: Action
  User can refresh analysis or start new scan

15. QUICK START GUIDE
================================================================================

Installation:
  1. pip install -r requirements.txt
  2. Place YOLOv8 model in model/best.pt
  3. python app.py

Usage:
  1. Open http://localhost:5000
  2. Click "Start Camera" or upload image
  3. Capture/upload rice leaf image
  4. View instant AI classification
  5. Use refresh or new scan controls

Troubleshooting:
  • Camera not working? Check permissions
  • No results? Verify model file
  • Styling issues? Clear browser cache
  • Performance slow? Check network connection
